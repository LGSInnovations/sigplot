Plot

Experiment with the data:

Notice that the data plotted is y, a JavaScript array.

Change the sin function in the for loop to see real-time changes in the plot.

To change the range of x values plotted, modify the parameter start and the value it is inremented by.

Experiment with the appearance:

To change the color of the plot, change the word orange to the name of any basic color in quotes, or a hexcode.

You can also change the values of the variables listed in brackets where the plot is declared and see how they affect the plot.

You can change the value of line from 3 to either 0 or "none" to show no line, 1 or "vertical" to show vertical lines, 2 or "horizontal" to show horizontal lines, or back to 3 or "connecting" for connecting lines.

Underneath the line argument, you can add the argument symbol with the following possible values: 1 or "pixel" (for small dots), 2 or "circle" (for circles), 3 or "square" (for squares), 4 or "plus" (for plus signs), 5 or "x" (for x's), 6 or "triangle" (for upward pointed triangles), 7 or "itriangle" (for downward pointed triangles), 8 or "hline" (for short horizontal lines) or 9 or "vline" (for short vertical lines).
